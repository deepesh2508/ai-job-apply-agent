# AI Job Apply Agent 🤖

AI-powered assistant to automatically fill LinkedIn Easy Apply forms using LLM + browser automation.

## Features

* Dynamic form detection
* Rule-based + AI hybrid answers
* Multi-step form handling
* Resume upload
* Optional auto-submit

## Tech Stack

* Python
* Playwright
* OpenAI API

## Setup

pip install -r requirements.txt
playwright install

## Run

Start Chrome:

chrome.exe --remote-debugging-port=9222

Open LinkedIn → Easy Apply

Run:

python main.py

## Note

Use responsibly. Avoid bulk automation.
